# AgenteFinanceiro
Agente de IA que faz análises Fundamentalistas e de Valuation

Para Utilizar o agente você precisa inserir uma chave de API do Google gemini no arquivo [config.py](config.py) dentro das aspas indicadas.

Para rodar o agente basta ter o Python instalado no PC conexção com a internte e executar o arquivo main.py

Sugestão de nome para o agente: AgFin
